Компьютерый код

  • Тег <code>(HTML Code Element) предназначен для выделения чисти компьютерного кода.

    • Следующий код выводит сообщение:<code> alert('Привет, друг');</code>

  • Тег <samp> (HTML Samp Output) определяет результат вывода компьютерной программы или скрипта.

    • <samp>Результат вывода компьютерной программы</samp>

  • Тег <kbd>(HTML Keyboard Input Element) определяет вводимые пользователем нажатия клавиш или текст, набираемый на клавиатуре.

    • Чтобы скопировать текст нажмите <kbd>Ctrl</kbd>+<kbd>c</kbd>

  • Тег <var>(HTML Variable Element) определяет математичесие / переменные величины.

    • <var>x</var>=<var>x</var>+ 1

    ‡ Элементы <code>, <samp> и <kbd> по умолчанию представлены моноширным шрифтом (все знаки (кегельные площадки знаков) имеют одинаковую ширину), а переменные величины (тег <var>) браузеры отображают курсивом.

    <!DOCTYPE html>
                <html>
                    <head>
