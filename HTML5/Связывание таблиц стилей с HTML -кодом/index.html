Связывание таблиц стилей с HTML-кодом

Подключение с использованием тега <link>

Элемент <link> определяет связь между документом и внешним ресурсом.

Элемент <link> размещается всегда внутри элемента <heaad>.

        <link  rel = "stylesheet" href =  "styles.css"> /* синтаксис HTML 5 */
  1. rel = "stylesheet" - определяет отношение между текущим документом и связатнным документом на который ведёт ссылка (в нашем случае- таблица стилей).
  2. href - задает URL-адрес местонахождения внешнего файла.

Подключение с использованием правила @import

CSS - правило @import используется для импорта содержмиого CSS файла в текущую таблицу стилей. Это правило должно предшествовать всем другим видам правил, за исключением привила @charset, которое задаёт кодировку символов, используемую в таблице стилей.

Основное отличие правила @import заключается в том, что с использованием этого правила вы можете присоединить одну внешнюю таблицу стилей к другой.
